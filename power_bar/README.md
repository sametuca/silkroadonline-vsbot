# Power Bar Detection

## How to Use

1. Wait for your power bar to be **COMPLETELY FULL** in-game
2. Take a screenshot (Windows + Shift + S)
3. Select **only the full bar** (small area, just the bar itself)
4. Save the image to this folder as **`bar_full.png`**
5. Enable "Power Bar Detection" in the bot GUI

## Tips

- Keep the screenshot small (only the bar)
- Don't include UI elements or background
- Make sure the bar is 100% full when taking the screenshot
- Use PNG format for best results

## How It Works

The bot will check for the bar image every 15-20 seconds (adjustable). When it finds a match, it automatically presses TAB to activate your power.

# Güç Barı Tespiti (Türkçe)

## Nasıl Kullanılır

1. Oyunda güç barınızın **TAMAMEN DOLMASINI** bekleyin
2. Ekran görüntüsü alın (Windows + Shift + S)
3. **Sadece dolu barı** seçin (küçük alan, sadece barın kendisi)
4. Resmi bu klasöre **`bar_full.png`** olarak kaydedin
5. Bot arayüzünde "Güç Barı Tespiti" seçeneğini aktif edin

## İpuçları

- Ekran görüntüsünü küçük tutun (sadece bar)
- UI elementlerini veya arka planı dahil etmeyin
- Ekran görüntüsü alırken barın %100 dolu olduğundan emin olun
- En iyi sonuç için PNG formatı kullanın

## Nasıl Çalışır

Bot 15-20 saniyede bir (ayarlanabilir) bar resmini arar. Eşleşme bulduğunda, gücünüzü aktive etmek için otomatik olarak TAB tuşuna basar.
